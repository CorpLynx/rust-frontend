# Documentation Directory

This directory contains technical documentation for the Prometheus project.

## Files

- **ARCHITECTURE.md** - System design, component overview, and data flow diagrams
- **CHANGELOG.md** - Version history and release notes
- **prometheus-cli.1** - Man page for the CLI application (install with `man ./prometheus-cli.1`)

## Main Documentation

For user-facing documentation, see the root directory:

- **../README.md** - Main documentation and usage guide
- **../INSTALL.md** - Installation instructions
- **../QUICKSTART.md** - Quick start guide
- **../STRUCTURE.md** - Repository organization

## Contributing

When adding new technical documentation:

1. Keep user-facing docs in the root directory
2. Keep technical/developer docs in this directory
3. Update this README.md to list new files
4. Update the main README.md if it affects users
